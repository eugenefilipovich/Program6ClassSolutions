Carousel Instructions

The purpose of the carousel is to rotate images by pressing a button. To do this you'll need to:

  • Use Angular's two way data-binding to display a current image
  • Bind an ng-click event to the next/previous buttons
  • In the function for the click next event:
    • Increase the current value by 1. If its greater or equal to the length of the image array, set it to 0.
    • Angular does the rest!
    • Do the opposite for the Previous function
